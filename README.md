## 💡 About This Project

I'm not a software developer. I'm someone who got curious about AI tools and decided to see how far I could go building real apps without writing code manually.

The answer: pretty far.

Each of these apps is live, functional, and built entirely through conversations with AI tools — describing what I want, iterating on the output, and deploying with GitHub + Vercel.

**Focus areas:** Indian culture, Hindu spirituality, kids' learning, and everyday utility apps for Indian users.
